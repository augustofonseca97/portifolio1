Projeto Cafeteria

Neste projeto, desenvolvemos um site para cafeterias focado exclusivamente na versão desktop. A interface conta com áreas interativas utilizando efeitos hover em CSS, além de um menu de navegação estruturado para facilitar o acesso às seções desejadas. Para finalizar, incluímos links internos para as redes sociais e utilizamos o código de incorporação do Google Maps para exibir a localização exata do estabelecimento de forma visível e prática.
